Created2022-12-03|Web
搭建个人博客一个傻瓜的办法，可以自动生成博客网站，然后将其托管在github.io，这样不用自己租云服务器和购买域名，爽歪歪 1.安装nodejs下载安装包自己安装https://nodejs.org/en/ 2.安装HexoHexo 是一个能自动生成网站代码的框架（巨好用） npm install -g hexo-cli （npm 是 nodejs 的命令） 检查 Hexo 是否安装成功：hexo -v 查看 Hexo 帮助 :hexo –help 3.生成网站代码初始化新建项目文件夹，并进入文件夹内： mkdir Blog cd Blog 初始化项目（必须进入到项目文件夹里面后，执行此命令） hexo init 启动网站hexo s 浏览器打开 http://localhost:4000 能看到生成的网站页面 4.更换网站主题官网下载主题 https://hexo.io/themes/ clone主题代码到 本项目下的 themes/butterfly git clone https://github.com/jerryc127/hexo-theme-butterf ...
